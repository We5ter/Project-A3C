## Overview

> рҹҡҖ **AВіC** is an open-source project dedicated to building **fully autonomous AI cybersecurity agents** that can independently execute penetration testing, vulnerability assessment, threat hunting, and incident response.

AВіC leverages cutting-edge Large Language Models (LLMs) and multi-agent orchestration to create a new paradigm for security operations вҖ” where **AI agents autonomously plan, execute, and adapt** across the entire attack and defense lifecycle.
